# PawSteps Dog Walking Website

## Image Requirements

This website uses the following images that need to be added to the `images/` folder:

### Required Images

1. **logo.png** - Your business logo (recommended: 200x200px, PNG with transparent background)

2. **hero-dog.jpg** - Main hero image showing a happy dog or you with a dog (recommended: 800x600px)

3. **about-me.jpg** - Photo of you with a dog for the About section (recommended: 500x600px)

4. **client-1.jpg** - Client testimonial photo (recommended: 150x150px, circular crop)

5. **client-2.jpg** - Client testimonial photo (recommended: 150x150px, circular crop)

6. **client-3.jpg** - Client testimonial photo (recommended: 150x150px, circular crop)

### Where to Find Images

For temporary placeholder images while building your site, you can use:

- **Unsplash.com** - Free high-quality photos (search "dog walking", "happy dog", "pet owner")
- **Pexels.com** - Free stock photos  
- **Pixabay.com** - Free images

### Image Optimization Tips

- Compress images to reduce file size (use TinyPNG.com or similar)
- Use JPEG for photos, PNG for logos with transparency
- Optimize for web to ensure fast loading
- Consider using WebP format for even better compression

### Quick Setup

1. Download placeholder images from the sources above
2. Rename them according to the list above
3. Place them in the `images/` folder
4. Replace with your actual photos when ready

## Customization Guide

### Updating Business Information

1. Replace "PawSteps" with your business name
2. Update phone number and email in the contact section
3. Update the location in the contact section
4. Modify prices in the pricing section to match your rates
5. Customize service descriptions and offerings
6. Replace the about section content with your own story and credentials

### Color Scheme

- Primary Color: #e74c3c (red-orange)
- Secondary Color: #f39c12 (orange)
- Text Color: #2c3e50 (dark blue-gray)
- To change colors, search for these hex codes in `styles.css`

### Fonts

- Currently using Poppins from Google Fonts
- To change fonts, update the Google Fonts import in `index.html` and the font-family in `styles.css`
